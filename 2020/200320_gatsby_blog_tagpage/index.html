Gatsbyブログにタグ機能を追加

GatsbyReact

March 20, 2020

はじめに

前記事では、記事一覧ページを複数ページに分割しました。
今回の目的は、

  • 記事に(複数の)タグを追加できるようにし、タグ毎に記事一覧ページを追加する

です。
(ジャンルも同様の作業で追加できます)

どのようにタグを追加するかというと、Markdownで記述するブログ記事の先頭に記述するメタデータを以下のように拡張します。 

---
title: タイトル
date: 2020-01-01T00:00:00.000000
description: 概要
tags: タグ1 タグ2 タグ3
---

tagsという分類を追加し、空白で区切ることによりタグを複数個設定できるようにします。

記事一覧ページに表示される記事にタグが表示されるようにする

記事タイトルと時間の間にタグが並ぶように表示させ、クリックするとそのタグの一覧ページに飛ぶようにします。
この部分のテンプレートはsrc/components/page.jsなので、以下のように修正します。 

import React from "react";
import { Link, } from "gatsby";
import { FontAwesomeIcon } from '@fortawesome/react-fontawesome';
import { faTag } from '@fortawesome/free-solid-svg-icons';
import { genTags, } from "../utils/format";

const Post = node => {
  const title = node.frontmatter.title || node.fields.slug;
  // node.frontmatter.tagsは文字列なので、これを文字列のタグの配列にパースします
  const tags = genTags(node.frontmatter.tags);
  return (
    <article>
      <header className="index-post-header">
        <h3>
          <Link style={{ boxShadow: `none` }} to={node.fields.slug}>
            {title}
          </Link>
        </h3>
        <div>
          {tags.length > 0 && <FontAwesomeIcon icon={faTag} />}
          {tags.length > 0 && tags.map((tag) =>
            <Link style={{ boxShadow: `none`, marginLeft: "10px"}} to={`tags/${tag}`}>
              {tag}
            </Link>
          )}
        </div>
        <small>{node.frontmatter.date}</small>
      </header>
      <section>
        <p
          dangerouslySetInnerHTML={{
            __html: node.frontmatter.description || node.excerpt,
          }}
        />
      </section>
    </article>
  );
};

export default Post;

ここでnode.frontmatter.tagsを使用していますが、もちろんクエリを取得して置く必要があります。
つまりPost関数を呼び出しているテンプレートページで取得します。
私の場合は、src/index.jsとsrc/templates/index.jsに取得する記述を追加しました。

因みに、../utils/formatからインポートしているgenTags関数は、空白で区切られたタグ(文字列)が含まれる文字列からタグの配列を生成するユーティリティ関数です。 

// タグ文字列からタグ配列を生成
const genTags = (tags) => {
  if (tags == undefined)
    return [];
  const _tagList = tags.split(/\s+/);
  const tagList = _tagList.filter(t => t.length > 0).sort();
  return tagList;
}

タグページの追加

先ほどの記述でタグをLinkコンポーネントで追加しましたが、これはtags/タグ名ページへのリンクになっています。

このページを作成していきます。

前回の記事と同じように、gatsby-node.jsのcreatePages関数内に以下の処理を追加します。
追加する処理を整理すると、

  1. 重複しない全てのタグを格納される配列を用意する
  2. 全ての記事に対してタグを取得し、先ほどの配列に追加する
  3. 全てのタグが格納された配列に対してループ処理でページを生成する

となります。 

  const tagPage = path.resolve(`./src/templates/tag-page.js`);
  /* 省略 */

// Create tab pages
  let tagList = posts.reduce((accumulator, post) => {
    const tags = genTags(post.node.frontmatter.tags);
    return retTagList(accumulator, tags);
  }, []);
  tagList.forEach((tag, index) => {
    createPage({
      path: `tags/${tag}`,
      component: tagPage,
      context: {
        tag: tag,
        index: 1,
      }
    });
  });

retTagList関数は既存タグリストと追加タグリストを入力とし、重複するものを削除しタグリストを返すユーティリティ関数です

const retTagList = (prevTagList, addTagList) => {
  const concatTagList = prevTagList.concat(addTagList);
  const tagList = Array.from(new Set(concatTagList));
  return tagList;
};

終わりに

これで、タグ機能が追加されました。
ただタグページに表示される記事の上限がなく、全ての記事が表示されてしまうので、そこを何とかしたい感じはあります。(まだ記事数が少ないので問題ないですが)
